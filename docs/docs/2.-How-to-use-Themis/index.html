How to use Themis

Themis is made to be as simple as possible:

  • Build and install Themis.
  • Pick a use-case from simple objects guide.
  • Import a library into your high-level language of choice.
  • Choose object mode appropriate for you.
  • Follow the docs and examples, you're near the finish line already.
